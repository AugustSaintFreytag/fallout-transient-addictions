# Transient Addictions

A mod for Fallout: New Vegas / Tale of Two Wastelands that introduces a time and chance based natural recovery from addictions. If the player is in good health and has not taken the item in a few days, there is a random chance (based on endurance and luck) to heal an addiction on its own. Written in NVSE, extremely lightweight, ESPLess, safe to remove at any time.
